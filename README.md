# 프로젝트 정보

## Install & Run

- 😘Front-End

```jsx
cd ssalog_front
npm install
npm run serve
```

- 🏗Back-End

```jsx
cd ssalog_backend
mvn package
cd target
java -jar SSalog_ver1.0-0.0.1-SNAPSHOT.jar
```

## Required

- Maven (Build tool)
- Git
- Java
- Mysql
- MongoDB
- Redis
